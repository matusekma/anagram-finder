### Home assessment for a job application.  
The program finds all anagrams of the user's input word in a
dictionary. The algorithm is case-insensitive and ignores whitespaces.  
The algorithm works in constant time after sorting the input word's characters.  
Tests are written using Jest. Statement, Branch, Function and Line Coverage are all 100%.

Intalling dependencies:  
`npm install`  
Run CLI app:  
`npm run start`  
Run tests:  
`npm run test`  
Run tests with coverage:  
`npm run test-with-coverage`
